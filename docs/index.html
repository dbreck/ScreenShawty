Features

Everything you need, nothing you don't.



Screenshot Controls

  • Save location — Desktop, Downloads, Screenshots, Clipboard, or any custom folder
  • Format — PNG, JPG, HEIC, PDF, TIFF, GIF
  • Window shadow — toggle on/off
  • Floating thumbnail — toggle the preview overlay
  • Include date — toggle date in filenames
⇩

Clipboard Shrinking

  • Shrink clipboard image — resize and compress in one click
  • Auto-shrink — automatically shrink every image that hits your clipboard
  • Global shortcut — ⌃⌥⌘S to shrink from anywhere
  • Configurable — max width/height, output format, quality, metadata stripping
